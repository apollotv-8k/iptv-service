How to Install Apollo TV on iOS (iPhone & iPad)

Because iOS does not allow APK installation, you must use a third-party app installer.

Step 1 — Install a trusted iOS sideload app

We recommend using Scarlet or Sideloadly.

Download Scarlet: https://usescarlet.com/

Step 2 — Download the Apollo TV IPA

Use the link: https://apollotv-8k.com/apollo.ipa (replace with your IPA link if different).

Step 3 — Install via Scarlet or Sideloadly

Open Scarlet → Import IPA → Install. On Sideloadly → Select IPA → Connect iPhone → Start installation.

Step 4 — Trust the app

Go to Settings → General → VPN & Device Management → Trust the developer.

Step 5 — Open Apollo TV

Launch the app and log in to start streaming live channels and movies.
